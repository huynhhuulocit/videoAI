# Skills

Thư mục này dùng để bổ sung tài liệu skill cho VideoAI.

## Skill hiện có

- [VideoAI Frontend Dashboard](./videoai-frontend-dashboard/SKILL.md): Skill hướng dẫn build/review frontend và dashboard hiện đại cho VideoAI.

## Skill dự kiến

- Skill tạo prompt.
- Skill phân tích sản phẩm.
- Skill phân tích hình ảnh/video mẫu.
- Skill tạo video.
- Quy tắc input/output của từng skill.

## Ghi chú

Các skill trong thư mục này là project-local skill. Nếu muốn Codex tự động discover như skill hệ thống, có thể copy skill cần dùng sang `$CODEX_HOME/skills`.
