# AGENT

Thư mục này dùng để bổ sung tài liệu AGENT cho VideoAI.

## File điều khiển chính

- [AGENTS.md](../../AGENTS.md): Quy tắc vận hành chính cho agent/dev khi làm việc trong repo.

## Nguyên tắc chính

- Luôn đọc `AGENTS.md` trước khi implement task không tầm thường.
- Nếu code thay đổi logic đã được mô tả trong `docs/`, phải update docs liên quan trong cùng thay đổi.
- Nếu user update docs trước, agent cần đọc docs liên quan rồi implement theo docs.
- Nếu code và docs conflict, agent phải làm rõ hoặc cập nhật để đồng bộ.

## Nội dung có thể bổ sung sau

- Prompt vận hành agent theo từng loại task.
- Checklist chi tiết cho backend/frontend/AI workflow.
- Quy tắc review PR.
- Quy tắc tạo hoặc cập nhật skill/hook.
